 Static  Website Hosted on AWS

 
  Deployed a static business website for Kanksh Global Services using AWS S3 and Route 53. Designed a fully responsive interface using HTML & CSS and configured custom domain DNS records with Route 53. Demonstrated hands-on experience in cloud hosting concepts. 

 

 Project Overview
This project demonstrates how to deploy a fully static website using Amazon Web Services (AWS). The website was created for a business (Kanksh Global Services) and is hosted using S3 (Simple Storage Service) for storage and Route 53 for DNS management, making it publicly accessible through a custom domain.

 

 

 Key Features
Static content hosted securely on AWS S3
Custom domain connected using AWS Route 53
Fast and reliable global access
Low-cost, scalable hosting
Professional business layout
 Steps Taken

Created a responsive static website using HTML, CSS, and JavaScript.
Uploaded website files to an S3 bucket configured for static website hosting.
Registered domain name and configured Route 53 to point to the S3 bucket.
Verified domain setup with DNS records.
Successfully launched the site for public access.
